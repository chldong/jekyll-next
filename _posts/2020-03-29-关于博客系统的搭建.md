# 关于博客系统的搭建等相关知识收集

## 关键步骤

### 本地安装Jekyll

```bash
~ $ gem install bundler jekyll
~ $ jekyll new myblog
~ $ cd myblog
~/myblog $ bundle exec jekyll serve
# => Now browse to http://localhost:4000
```

### 本地安装Jekyll Theme

Jekyll Theme官网找到自己喜欢的主题，Demo查看效果；
Donwload主题源码，解压到本地并进入安装目录；

```bash
bundle install
bundle exec jekyll server
# => Now browse to http://localhost:4000
```
